1. Πως μπορούμε να κάνουμε ένα Request σε ένα εξωτερικό API χωρίς να 
γνωρίζει ολόκληρο το url μας;

-Χρησιμοποιώντας Referrer-Policy sto Header.

2. Αν κάνω ενα GET request και ζητήσω ενα jpeg image, το response τι 
content-type πρεπει να εχει;

- image/jpeg.

3. Πως μπορώ να κάνω ένα GET request και το περιεχόμενο του να 
κρατηθεί στην cache για 1 μέρα;

- Cache-Control: public,max-age=86400

4. Μπορώ να έχω μια εφαρμογή που κάνει μόνο GET requests;

- Ναι.

5. Μπορώ να έχω μια εφαρμογή που κάνει μόνο POST requests;

- Ναι.

6. Τι status code περιμένω από ένα API αν δεν έχω στείλει ένα 
υποχρεωτικό πεδίο;

- Κάποιο status code της κατηγορίας 400 και ανάλογως το API configuration και κάποια παράμετρο μαζί με αυτό.

7. Τι status code περιμένω από ένα API αν δεν υπάρχει επικοινωνία με τη 
βάση;

- Κάποιο status code της κατηγορίας 500 και ανάλογως το API configuration και κάποια παράμετρο μαζί με αυτό.

8. Μπορώ να κάνω Login με GET request;

- Ναι αλλα δεν είναι ασφαλές.

9. Ποια είναι η διαφορά ανάμεσα στο PUT και στο POST;

- H PUT request έχει ακριβώς το ίδιο αποτέλεσμα όσες φορές και αν καλεστεί.
Ενώ αν καλέσουμε το POST request πολλές φορές θα δημιουργεί νέα δεδομένα κάθε φορά.
Η PUT χρησιμοποιείται συνήθως για την αντικατάσταση πληροφοριών σε ένα συγκεκριμένο URL με κάτι άλλο.
Η POST χρησιμοποιείται συχνά για να στείλει δεδομένα που δημιουργούνται από το χρήστη στον Server.


10. Ποια μέθοδο θα πρέπει να χρησιμοποιήσω για να «ανεβάσω» ένα 
αρχείο;
